# 📢 긴급 안내: 앱 실행 종료 현상 및 해결 방법

안녕하세요, **섭식이** 개발자입니다.

먼저, **지금 이 글을 보고 계신다면 회원님의 섭식이는 무사하다는 뜻입니다.** 안심하셔도 좋습니다. 😄

하지만 현재 일부 사용자분들께서는 앱 실행 시 강제 종료되는 현상을 겪고 계십니다. 해당 문제를 겪고 계신 분들과, 혹시 모를 상황에 대비해 해결 방법을 안내해 드립니다.

## 🚨 발생 현상 및 원인
현재 특정 버전(v1.0.2.3 ~ v1.0.2.4.1)에서 **'자동 업데이트' 프로세스 실행 중 앱이 종료되는 현상**이 확인되었습니다.

사용자 편의성을 극대화하기 위해 앱 실행 시 최신 버전을 즉시 로드하는 **'심리스(Seamless) 업데이트'** 기능을 도입했으나, **이 과정에서 구글 플레이 API의 보안 정책(Intent Lifecycle)과 일부 기기 환경 간의 충돌이 발생**한 것으로 파악되었고 조치한 상황입니다.

## 🛠️ 해결 방법 (v1.0.3 배포 전까지)
해당 이슈를 근본적으로 해결하고, **새로운 기능성을 탑재한 v1.0.3 버전**을 준비 중입니다.
다만 현재 문제가 발생한 버전에서는 앱 내부 로직만으로는 복구가 불가능한 상태입니다.

번거로우시겠지만, **v1.0.3 배포 전까지는 직접 구글 플레이 스토어를 방문하여 수동 업데이트**를 부탁드립니다.

1. 스마트폰의 **[구글 플레이 스토어(Play Store)]** 앱 실행
2. 검색창에 **'섭식이'** 검색
3. **[업데이트]** 버튼 터치
