# 📢 섭식이 업데이트 소식

안녕하세요! 건강한 습관의 시작, **섭식이**입니다.
사용자분들의 소중한 의견을 반영하여 더 편리해진 섭식이를 만나보세요.

## ✨ 이번 업데이트의 주요 변경 사항

### 🚀 1. 데이터 동기화 속도 대폭 개선
식품 데이터베이스를 동기화하는 속도가 **3배** 더 빨라졌습니다. 이제 기다림 없이 쾌적하게 데이터를 이용하세요!

### 🌍 2. 글로벌 언어 지원 확대
이제 **인도네시아어**와 **브라질 포르투갈어**를 정식 지원합니다.
전 세계 친구들에게 섭식이를 추천해 보세요.

### 🛠️ 3. 입력 오류 수정 (소수점 문제)
브라질, 유럽 등 일부 국가 표기법 사용 시, 키와 몸무게 입력에서 **콤마(,)**가 제대로 저장되지 않던 문제를 완벽하게 해결했습니다.
이제 `171,3`과 같이 편하게 입력하셔도 정확하게 기록됩니다.

### 🎨 4. 기타 개선
* 작은 화면에서의 UI 가독성을 개선했습니다.
* 앱 안정성을 높였습니다.
