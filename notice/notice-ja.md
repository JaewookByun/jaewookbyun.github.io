# 🥑 v1.0.5 アップデート：海外ユーザー待望の「食事手動入力」機能を追加！

こんにちは、**Supsiki**の開発者です。

これまでバーコードや検索で見つからない食品を食べたとき、記録に困ったことはありませんか？
特に韓国の食品データベースにない食品を召し上がる海外ユーザー様の不便を解消するため、ついに**「手動入力（Manual Entry）」**機能をサポートしました！

## ✅ リリースノート

### 1. 検索できない食品？これからは「手動入力」で解決 ✍️
これまでは**バーコード、韓国の品目報告番号、韓国語検索**でのみ記録が可能で、それ以外の食品は外部アプリで記録してヘルスコネクト経由で読み込む必要がありました。
これからは、Supsikiアプリ内で直接栄養成分を入力して記録できます！

**💡 開発者が教える入力のコツ！**
* **正確に入力するなら：** 栄養成分表示を見ながら**[カロリー → 炭水化物 → タンパク質 → 脂質]**の順に入力すると最も正確です。
* **計算が面倒なら：** カロリー欄を空けておき、**[炭水化物 → タンパク質 → 脂質]**だけ入力してみてください。Supsikiが自動でカロリーを計算して入力します。
* **海外食品も安心：** パッケージの表示が**kJ（キロジュール）**単位でも心配いりません。入力時に自動的に**kcal**に変換されます。

### 2. ヘルスコネクト双方向連携 🔗
手動で入力した食事データは、**ヘルスコネクト（Health Connect）**を通じて他の健康管理アプリと完璧に連携されます。Supsikiで記録して、どこでもデータを確認しましょう。

### 3. 商品比較機能の改善 ⚖️
バーコードをスキャンして最大4つの商品を一目で比較できる機能をご存知ですか？この比較機能をより便利に使えるよう、少し改善を行いました。

### 4. 多言語対応の改善とバグ修正 🛠️
グローバルユーザーの皆様のために翻訳をより自然に修正し、アプリの安定性を高めるため細かなバグを修正しました。
